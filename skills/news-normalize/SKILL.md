# news-normalize

## 当前实现补充

本地已接入最近新闻发现能力，覆盖四个主题：

- 财经
- 政治
- 军事
- 科技

当前用法：

```bash
python3 core/news.py
```

## 目的

将新闻原文或新闻链接整理成适合后续排序和摘要的结构化候选结果。

## 适用场景

- 财经与政治新闻整理
- 市场相关新闻初筛
- 简报前的标准化处理

## 输入

- 新闻链接
- 新闻正文
- 多条新闻候选列表

## 输出

- 标题
- 来源
- 时间
- 主题标签
- 事实摘要
- 可能影响
- 待验证点

## 不负责

- 最终推送决策
- 长期任务创建
- 长期记忆写入
- 模拟交易执行

## 默认要求

- 优先分离事实和判断
- 标注信息是否需要进一步核验
- 多条新闻时尽量去重并合并相似事件
